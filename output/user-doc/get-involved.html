Join the Apache River Community¶

The Apache River project is being built by the open source community for the open source community. We truly welcome your input and contributions!

What we are looking for:¶

  • Bug fixes and code enhancement contributions
  • Source code contributions
  • Documentation assistance and contributions
  • Feature suggestions
  • Detailed and constructive feedback
  • Articles and white papers
  • Web site improvements
A recent initiative aims at revising JINI specifications:
"The idea is to prioritize the modernization of the River specification into a set of language and transport agnostic architectural principles. River currently supports architectural concepts like discovery, events, proxies and more! In reality, both the implementation language and communication transport are minor details."

Help the community in this exciting modernization plan!

Read the above thread to get a context, use JIRA to find a task you are interested in and follow our source code guidelines to submit changes to specifications. Thank you!

How do I contribute?¶

  • To discuss Apache River topics check out the mailing lists.
  • Bugs and other issues can be posted in the RIVER issue tracker. See more info here.
  • Learn how to deal with Web site related issues
